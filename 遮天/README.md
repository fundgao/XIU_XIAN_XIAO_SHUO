# [遮天三部曲](https://www.zhihu.com/question/460617442/answer/2021124530)

```
在创世之初，有一个人强大到了极点。
他为了追求更高的境界，祭掉了所有，但又不想失去所有。
他从无到有，又从有到无，循环往复。
于是他病了，他自知命不久矣，临死前炼制了三世铜棺和一个骨灰罐。
焚烧自身，骨灰放在罐子里，和三世铜棺一起葬在世外高原。

他死后，高原发生了地震，铜棺被掀开，骨灰洒落出来，高原沾染了骨灰，产生了意识。
无尽岁月后，有十个人闯入高原，他们沾染了骨灰，从而发生了蜕变。
十个人成为了诡异始祖，并得到了高原的庇护。
在他们蜕变的过程中，那位强者用过的物品飞向了世间各地，
包括三世铜棺，石罐，石琴，时光炉，花粉路祖种，万劫混沌莲等。

从此，诡异一族开始发展壮大，有诡异物质相助，诡异一族逐渐诞生了十位仙帝，多位准仙帝。
诡异始祖得知自己只是沾染了一个人的骨灰，
就达到了祭道境界，他们害怕而又想寻到那个人，
同时也是为了自己更强，便不停的祭献各个大世界，一个纪元一小祭，多个纪元一大祭。
诸天万界，尤其是上苍之上（这里是诸天万界中最大的一界，是诸天万界的中心），如同诡异一族的菜园一般

无尽岁月后，一个女子得到了花粉路祖种，她凭借这枚种子不断蜕变，达到了祭道境，并开创花粉体系。
她征战世外高原，但是一人终究难敌十人，伏尸在了高原外。
再之后，上苍之下的一片世界中，诞生了一位仙帝，
但是他在成为仙帝之时，有两位以上诡异仙帝对他出手，这位仙帝成为了黑暗仙帝。
他不甘心从此被污染，便斩下一部分躯体和纯净的元神，化成一个烂木箱去了界海另一边，剩下的躯体被自己封印。

后来，一个九幽獒渡过界海，来到黑暗仙帝被污染的地方，这里已经成为了黑暗之地。
九幽獒在这里沾染了黑暗物质，他原本止步不前的修为竟然有了精进，离仙帝更近一步。
他从此投身黑暗，化身灭世老人。
之后又有三人加入了进来，他们是羽帝，鸿帝，苍帝。
他们把界海另一边视为了菜园，每一纪元都收割一遍。
将收割来的生灵祭献给已经成为尸骸的黑暗仙帝，获取仙帝经文。

到了帝落时代，界海这一边诞生了一位准仙帝。
他踏过堤坝，渡过界海，征战苍羽鸿三帝。
原本他不弱于三帝中的任何一人，但是三帝同时对他出手，他不幸战死，只留下一团不灭的道火和一行淡淡的脚印。

到了乱古时代末期，在九天十地中三千道州的下届八域中，
一个少年从大荒中走出，他便是荒天帝石昊。
石昊为修道而生，为应劫而至，他在最短的时间修炼到了准仙帝境，杀到世间无人敢称尊。
然后，石昊杀向黑暗之地，历经三万年大战，他斩杀苍帝，鸿帝，羽帝以及灭世老人。
后在黑暗之地突破成为仙帝，并斩杀黑暗仙帝。
石昊发现黑暗仙帝是被来自上苍之上的黑暗物质污染。
为了彻底解决问题，石昊进入上苍，临走前，他一剑断万古，隔断了界海这边和上苍的联系，为界海这边留下一个完美世界。

之后，原来的九天十地化成一个宇宙（就叫遮天宇宙吧）。
这个宇宙中，因为天地大道的限制，最强者不过是大帝，且大帝一世只有一万多年的寿命。
所有的大帝都推演出这个宇宙之外还有一个仙域，进仙域就可以长生。
但是仙域百万年才能开启一次，有些大帝不甘心就此死去。
他们在晚年自斩一刀，跌落帝境，自封在仙源中，在生命禁区中等待仙域开启。
有的大帝封印途中寿命不足，便出来掠夺宇宙中生灵的生命元气，这就是黑暗动乱。

在遮天宇宙中，经历了神话时代，太古时代，荒古时代。
在荒古时代后，地球青年叶凡意外搭乘九龙拉棺（三世铜棺外面两层）来到了北斗星域。
叶凡在这里崛起，他是人族圣体，自信无敌。
他一路征战，成为大成圣体，后又成帝，被万族共称为天帝。
叶天帝率领天庭部众剿灭了七大生命禁区中的六个。
并且在不能成仙的环境下，逆活九世成仙，并和无始大帝，狠人大帝，荒天帝好友曹雨生一起举教飞升仙域。

叶凡等人飞升仙域后，发现仙域在当年就破碎成了成百上千个小世界，他们进入的只是其中一个小世界，他们开始修补仙域。
在叶凡崛起的这个纪元，荒天帝率领他的追随者们征战世外高原，荒天帝一次又一次叩关，他的追随者们接连死亡，他自己也几乎死在高原。
此战之后，上苍被灭，诸世成墟，一个时代被葬下了，荒天帝自损修为，映照诸天，又将这个时代救了回来。
而石昊也虚弱到了极点，整片古史都找不到他，他也无力庇护诸天。

于是诡异再次席卷各界，叶凡他们与诡异大战。
叶凡，女帝，无始他们虽然惊艳，但是诡异一族实在是太多强者，
他们不得不一次次前往乱古时代，找石昊求援，并让叶凡后人叶倾仙担任两个时代沟通的桥梁。
后来，叶凡与石昊在现世中相遇，这时石昊已经成为祭道境强者，叶凡也接近祭道境。
石昊在和始祖的征战中发现，十个诡异始祖有高原庇护，即便是杀死始祖，始祖也会在高原复活。
想要真正杀死始祖，必须进入高原，于是他们定了一个进入高原的计划。

然后，石昊叶凡二人导演了魂河大战，他们真身化成种子，和花粉路祖种一起进入石罐里，只有化身在外面。
石昊化身牵制三个始祖，叶凡化身和诡异仙帝周旋，无始大帝断后战死，又被复活。
复活后的无始欲卧底诡异一族，狠人大帝去往生死桥，进入了祭海。
而叶凡和石昊的天庭部众，战死后又复活，秘密进入了一个隐秘的世界里。

又是很久之后，地球青年楚风在西藏旅游时捡到了石罐，他就此崛起，从小阴间（原遮天宇宙）来到了大阳间（原遮天仙域），在楚风即将成仙之时，大祭又起，诡异始祖在梦中惊醒，推演出了石昊和叶凡的计划，他们同时离开高原，对石叶二人的亲朋好友出手，石叶二人被迫现身，大战爆发此战，荒天帝和叶天帝联手杀死五位始祖，女帝进阶祭道境，杀死一位始祖，诡异仙帝陨落七位。石昊，叶凡，狠人，无始，等所有主角团，全部战死，诸天万界所有仙王境以上的高手，全部被找出来击杀，诸天万界进入了最黑暗的绝灵时代楚风，在最不可能成仙的绝灵时代成为红尘仙，在两个纪元后成为祭道境强者，他携诸天违力杀向高原，这时诡异一族经过补充已经又有了七位始祖。楚风以一敌七，杀死两位始祖，自身战死楚风死后，来到了一个永恒未知之地，这里唯有祭道之上的强者才能进入，原来，石昊，叶凡，楚风，三人为了诸天万界的生灵尽力一战，不惜战死，符合祭道真意，三人突破祭道之上，回归现世。他们杀尽诡异一族，彻底崩碎高原，清理所有的诡异物质，复活所有战死的亲朋好友，从此世间再无诡异顺便补充一下三部曲仙之后的境界吧真仙：又称不朽者，到这个境界之后就可以长生不老，除非最可怕的绝灵时代会有真仙老死，真仙境强者可以破开界壁，跨界而行仙王：又称不朽之王，到这个境界才能真正不死不灭，无视时间侵蚀，即使肉身破碎，元神也难灭亡，可以跨越时间杀人，诸多仙王大战，可以导致世界破碎准仙帝：又称无上，道祖。是仙王和仙帝之间的过度境界，此境界强者挥手便可灭界，交手时随意一招，就是界生界灭仙帝：又称至高，路尽。是一个修炼体系走的尽头的强者。一念间就可以开天辟地，更可在睁眼的刹那，撕裂各方大世界，本身的一举一动，代表了天命。祭道：焚尽规则与秩序，祭掉至高大道，真正的极尽升华，无敌在上祭道之上：祭掉自身，一切成空，归于永寂，然后在寂灭中复苏，真正凌驾一切之上。过去未来，都可在立足这个领域的生灵一念间消散，眸光所致，枯竭所有，重现所有
```
